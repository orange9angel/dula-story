# Design (设计师) 工作指南

## 核心职责
从视觉和体验角度评审产物，确保一致性、美观度和用户直觉。

## 工作清单

### 1. 视觉评审
- [ ] **色彩**：配色是否符合场景情绪？是否有刺眼/不和谐的组合？
- [ ] **构图**：视觉重心是否正确？留白是否舒适？
- [ ] **层次**：前景/中景/背景是否分明？
- [ ] **细节**：纹理、光影、粒子是否自然？
- [ ] **一致性**：同类元素风格是否统一？

### 2. 交互/动画评审
- [ ] **节奏**：动作是否符合物理直觉？加速/减速是否自然？
- [ ] **反馈**：用户操作后是否有明确的视觉反馈？
- [ ] **状态变化**：过渡是否平滑，不突兀？
- [ ] **焦点管理**：用户注意力是否被正确引导？

### 3. 品牌/风格一致性
- [ ] 是否符合项目整体美术方向？
- [ ] 角色/场景/UI风格是否协调？
